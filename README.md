## Getting Started

### Prerequisites

- A modern web browser (Chrome, Firefox, Safari, or Edge)
- [Git](https://git-scm.com/) installed locally
- A code editor such as [VS Code](https://code.visualstudio.com/)

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/CHRISTABELKWANING/FlyRank-Assignment.git
   ```

2. Navigate into the project folder:

   ```bash
   cd FlyRank-Assignment
   ```

3. Open `index.html` in your browser, or use a local development server:

   ```bash
   # Example with VS Code Live Server extension, or:
   npx serve .
   ```

## Development Practices

- **Conventional Commits** — commit messages follow the [Conventional Commits](https://www.conventionalcommits.org/) specification (e.g. `feat: add search filter`, `fix: correct mobile nav toggle`)
- **Semantic HTML** — meaningful elements (`header`, `main`, `section`, `nav`) for accessibility and SEO
- **Progressive enhancement** — core content works without JavaScript; interactivity is layered on top
